1 /*
2  * cLerpFunctionInterval.h
3  *
4  * Created on: 2012-05-30
5  * Author: dri
6  *
7  * This class is meant to emulate the capabilities of python LerpInterval class. It is by no
8  * mean a full implementation. It only contains what is needed to translate the Panda3D tutorials.
9  *
10  * CLerpFunctionInterval is templatized so that it can lerp a class (for instance, vector<double>).
11  * For a given class A, the following operators must be defined:
12  * A A::operator=(const A&)
13  * A A::operator-(const A&)
14  * are used to define the lerp range, such as `range = end - start'.
15  * A A::operator+(const A&)
16  * A A::operator*(double)
17  * each frame, the current A is computed such as `current = start + range * t'
18  * where t starts with 0 at the beginning of the interval and ends with 1.
19  * A A::operator<(const A&)
20  * is used to make sure `start <= current <= end' on the whole interval.
21  *
22  * Each frame, `current' is passed to the user-defined function LerpFunc along with
23  * a pointer to the user-defined data.
24  */
