1 /*
2  * cCommonFilters.h
3  *
4  * Created on: 2012-06-20
5  * Author: dri
6  *
7  * This class implements in C++ Panda3D python CommonFilters. I have tried to
8  * remain as close as possible to the python code. This is why structs are used
9  * instead of classes. Also, light-weight structs are used to simulate python
10  * positional parameters but the design could be further improved using the name
11  * parameter idiom. -dri
12  *
13  * Original python header:
14  *
15  * Class CommonFilters implements certain common image
16  * postprocessing filters.
17  *
18  * It is not ideal that these filters are all included in a single
19  * monolithic module. Unfortunately, when you want to apply two filters
20  * at the same time, you have to compose them into a single shader, and
21  * the composition process isn't simply a question of concatenating them:
22  * you have to somehow make them work together. I suspect that there
23  * exists some fairly simple framework that would make this automatable.
24  * However, until I write some more filters myself, I won't know what
25  * that framework is. Until then, I'll settle for this
26  * clunky approach. - Josh
27  *
28  */
